# ONCE_Javiera

## conocer mis credenciales.

1. vas a github.com e inicias sesion o registrate.
2. Despues vas repositorios y le das al boton verde que dice +new.
3. Posteriormente te mostrara una pagina donde podras configurar a profundidad tu repositorio, pero en esta ocasion solo le pondras el nombre y lo pondras para que sea publico.
4. Luego le das en el cuadro que dice codespaces.
